# Évaluateur d’Expressions Arithmétiques en C

Ce programme permet d’évaluer des expressions mathématiques avec :
+ - * / et parenthèses.

## Exécution

gcc main.c -o eval
./eval

## Exemple

Entrée : (2+3)*4=
Sortie : 20
